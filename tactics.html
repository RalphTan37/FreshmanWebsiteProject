    Tactics

    Forks

    A Fork is a tactic in which a piece attacks 2 or more pieces at the same time.

    If a piece that is forked moves away, then the piece forking can capture the other forked piece, vice versa.

    See GIF above for reference.


    Pins

    A Pin is a tactic in which a piece paralyzes a piece by attacking it, if that pieces moves away, then the piece behind it will be captured.

    The piece that is behind the paralyzed piece is either of higher value than the paralyzed piece or is unprotected.

    See GIF above for reference.


    Skewers

    A Skewer is a tactic in which a piece attacks two pieces in the same line.

    Very similiar to a Pin but in a Skewer, the higher value piece is attacked and the lower value piece is behind the higher value piece.
